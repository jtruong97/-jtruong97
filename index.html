About Me

Get To Know Me

I'm Jasmine Truong, a Full Stack Software Engineer

I grew up in San Francisco but recently relocated to Sacramento. I graduated with a Bachelor's in Biology, aspiring to become a Physician Assistant at the time. After graduation, I worked as a nurse's assistant and later as a medical assistant for approximately 3 years. As much as I loved working in healthcare, I realized it wasn't for me.

I spent some time exploring different career paths and fell in love with coding, and that was the moment I knew I wanted to be a software engineer. I joined a coding bootcamp and learned how to code my very own full-stack web application which equipped me with all the necessary tools such as JavaScript, Python, and React, as well as soft skills including resilience, communication, and learning from others. Amazonian, a full-stack application I developed using Flask backend and React frontend, is a clone of ‘Amazon’ that I am proud of. This project has refined my skills in database management, React state management, and taught me how to implement features such as AWS for image storage and OAuth with Google for user authentication. My favorite part of the development process is coming up with an idea, being able to execute it, and watching it come to life.

Outside of work, I love to travel to learn different cultures and try new cuisines. I am a huge foodie and always have a running list of new restaurants to try with my friends and family. To unwind at home, I enjoy binge watching TV shows, playing cozy games on my Switch, and experimenting with new recipes I find on TikTok.

My journey has been very rewarding, and I hope to end up at a company that shares the same values I do. A company that prioritizes creativity, collaboration, and continuous learning.
