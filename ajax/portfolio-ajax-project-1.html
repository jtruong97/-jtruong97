Stardew-n-dew

Project Info:

Stardew-n-dew is a small-scale clone of the website "Airbnb", with "Stardew Valley" twist. It is an online marketplace that allows users to book short-term and/or long-term home rentals.

As a user of Stardew-n-dew, you can create and curate listings with personalized details to ensure uniqueness. If needed, you can seamlessly edit your listing to reflect any changes or updates.

Stardew-n-dew allows users to share their thoughts on other listings through reviews. This will help users find the perfect rental for them as well as provide valuable insight for hosts to make improvements if necessary.
